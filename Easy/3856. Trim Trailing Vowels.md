# [3856. Trim Trailing Vowels](https://leetcode.com/problems/trim-trailing-vowels)

## Description

You are given a string `s` that consists of lowercase English letters.

Return the string obtained by removing **all** trailing **vowels** from `s`.

The **vowels** consist of the characters `'a'`, `'e'`, `'i'`, `'o'`, and `'u'`.

**Example 1:**

**Input:** s = "idea"  
**Output:** "id"  
**Explanation:**  
Removing `"id**ea**"`, we obtain the string `"id"`.

**Example 2:**

**Input:** s = "day"  
**Output:** "day"  
**Explanation:**  
There are no trailing vowels in the string `"day"`.

**Example 3:**

**Input:** s = "aeiou"  
**Output:** ""  
**Explanation:**  
Removing `"**aeiou**"`, we obtain the string `""`.

**Constraints:**

- `1 <= s.length <= 100`
- `s` consists of only lowercase English letters.

## My Solution

**JavaScript**

```js
const trimTrailingVowels = (s) => {
  const vowels = new Set(['a', 'e', 'i', 'o', 'u']);
  let i = s.length - 1;
  while (i >= 0 && vowels.has(s[i])) --i;
  return s.substring(0, i + 1);
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N)            | O(1)             |

**Python**

```python
def trimTrailingVowels(self, s: str) -> str:
    vowels = { 'a', 'e', 'i' ,'o' ,'u' };
    i = len(s) - 1

    while i >= 0 and s[i] in vowels:
        i -= 1

    return s[:i+1]
```
