# [3936. Minimum Swaps to Move Zeros to End](https://leetcode.com/problems/minimum-swaps-to-move-zeros-to-end)

## Description

You are given an integer array `nums`.

In one operation, you can choose any two **distinct** indices `i` and `j` and swap `nums[i]` and `nums[j]`.

Return an integer denoting the **minimum** number of operations required to move all 0s to the end of the array.

**Example 1:**

**Input:** nums = \[0,1,0,3,12\]  
**Output:** 2  
**Explanation:**  
We perform the following swap operations:

- Swap `nums[0]` and `nums[3]`, giving `nums = [3, 1, 0, 0, 12]`.
- Swap `nums[2]` and `nums[4]`, giving `nums = [3, 1, 12, 0, 0]`.

Thus, the answer is 2.

**Example 2:**

**Input:** nums = \[0,1,0,2\]  
**Output:** 1  
**Explanation:**  
We perform the following swap operations:

- Swap `nums[0]` and `nums[3]`, giving `nums = [2, 1, 0, 0]`.

Thus, the answer is 1.

**Example 3:**

**Input:** nums = \[1,2,0\]  
**Output:** 0  
**Explanation:**  
The array already satisfies the condition. Therefore, no swap operations are needed.

**Constraints:**

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 100`

## My Solution

**JavaScript**

```js
// Approach: Single pass using Two Pointers
const minimumSwaps = (nums) => {
  let result = 0;
  let left = 0;
  let right = nums.length - 1;

  while (left < right) {
    // Non-zeros belong on left side; advance pointer
    if (nums[left] !== 0) ++left;
    // Zeros belong on right side; advance pointer
    else if (nums[right] === 0) --right;
    else {
      // Reaching this point means left is 0 and right is non-zero
      // This is the swap condition; increment swap counter and adjust pointers
      ++result;
      ++left;
      --right;
    }
  }

  return result;
};
```

```js
// Approach: Count the number of zeros in the array. Then iterate backward
// through the last 'numZeros' positions of the array. Any non-zero found in this window requires one swap.
const minimumSwaps = (nums) => {
  let result = 0;
  let numZeros = 0;

  for (const num of nums) {
    if (num === 0) ++numZeros;
  }

  for (let i = nums.length - numZeros; i < nums.length; ++i) {
    if (nums[i] !== 0) ++result;
  }

  return result;
};
```
