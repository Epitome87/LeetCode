# [1582. Special Positions in a Binary Matrix](https://leetcode.com/problems/special-positions-in-a-binary-matrix)

## Description

Given an `m x n` binary matrix `mat`, return _the number of special positions in_ `mat`_._

A position `(i, j)` is called **special** if `mat[i][j] == 1` and all other elements in row `i` and column `j` are `0` (rows and columns are **0-indexed**).

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/12/23/special1.jpg)  
**Input:** mat = \[\[1,0,0\],\[0,0,1\],\[1,0,0\]\]  
**Output:** 1  
**Explanation:** (1, 2) is a special position because mat\[1\]\[2\] == 1 and all other elements in row 1 and column 2 are 0.

**Example 2:**

![](https://assets.leetcode.com/uploads/2021/12/24/special-grid.jpg)  
**Input:** mat = \[\[1,0,0\],\[0,1,0\],\[0,0,1\]\]  
**Output:** 3  
**Explanation:** (0, 0), (1, 1) and (2, 2) are special positions.

**Constraints:**

- `m == mat.length`
- `n == mat[i].length`
- `1 <= m, n <= 100`
- `mat[i][j]` is either `0` or `1`.

## My Solution

**JavaScript**

```js
// Approach: Record the number of 1's that occur for each row and column
const numSpecial = (mat) => {
  result = 0;
  const m = mat.length;
  const n = mat[0].length;
  const rowCount = Array(m).fill(0);
  const colCount = Array(n).fill(0);

  for (let r = 0; r < m; ++r) {
    for (let c = 0; c < n; ++c) {
      if (mat[r][c] === 1) {
        rowCount[r]++;
        colCount[c]++;
      }
    }
  }

  for (let r = 0; r < m; ++r) {
    for (let c = 0; c < n; ++c) {
      if (mat[r][c] === 1) {
        if (rowCount[r] === 1 && colCount[c] === 1) {
          result++;
        }
      }
    }
  }

  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(M \* N)       | O(M + N)         |

```js
// Approach: Brute-Force
const numSpecial = (mat) => {
  result = 0;
  const m = mat.length;
  const n = mat[0].length;

  for (let r = 0; r < m; ++r) {
    for (let c = 0; c < n; ++c) {
      if (mat[r][c] === 0) continue;

      let isGood = true;

      for (let row = 0; row < m; ++row) {
        if (row !== r && mat[row][c] === 1) {
          isGood = false;
          break;
        }
      }

      for (let col = 0; col < n; ++col) {
        if (col !== c && mat[r][col] === 1) {
          isGood = false;
          break;
        }
      }

      if (isGood) {
        ++result;
      }
    }
  }

  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(MN(M + N))    | O(1)             |
