# [938. Range Sum of BST](https://leetcode.com/problems/range-sum-of-bst)

## Description

Given the `root` node of a binary search tree and two integers `low` and `high`, return _the sum of values of all nodes with a value in the **inclusive** range_ `[low, high]`.

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/11/05/bst1.jpg)

**Input:** root = \[10,5,15,3,7,null,18\], low = 7, high = 15  
**Output:** 32  
**Explanation:** Nodes 7, 10, and 15 are in the range \[7, 15\]. 7 + 10 + 15 = 32.

**Example 2:**

![](https://assets.leetcode.com/uploads/2020/11/05/bst2.jpg)

**Input:** root = \[10,5,15,3,7,13,18,1,null,6\], low = 6, high = 10  
**Output:** 23  
**Explanation:** Nodes 6, 7, and 10 are in the range \[6, 10\]. 6 + 7 + 10 = 23.

**Constraints:**

- The number of nodes in the tree is in the range `[1, 2 * 10^4]`.
- `1 <= Node.val <= 10^5`
- `1 <= low <= high <= 10^5`
- All `Node.val` are **unique**.

## My Solution

**JavaScript**

```js
// Approach: Recursive DFS (without helper function) that also takes advantage of
// the Binary Search Tree nature of the tree, pruning out subtrees appropriately.
const rangeSumBST = (root, low, high) => {
  if (!root) return 0;

  // Prune right subtree if value is already too high
  if (root.val > high) return rangeSumBST(root.left, low, high);

  // Prune left subtree if value is already too low
  if (root.val < low) return rangeSumBST(root.right, low, high);

  // Value is within range of low to high; it and both subtrees contribute to the sum
  return root.val + rangeSumBST(root.left, low, high) + rangeSumBST(root.right, low, high);
};
```

| O(N)            | O(Height)        |

```js
// Approach: Recursive DFS (without helper function)
const rangeSumBST = (root, low, high) => {
  if (!root) return 0;

  const inRange = root.val >= low && root.val <= high;
  const leftSum = rangeSumBST(root.left, low, high);
  const rightSum = rangeSumBST(root.right, low, high);

  return (inRange ? root.val : 0) + leftSum + rightSum;
};
```

```js
// Approach: Perform a typical DFS, incrementing result any time the node's value
// falls within the range [low, high]
const rangeSumBST = (root, low, high) => {
  let result = 0;

  const dfs = (node) => {
    if (!node) return 0;

    if (node.val >= low && node.val <= high) {
      result += node.val;
    }

    dfs(node.left);
    dfs(node.right);
  };

  dfs(root);
  return result;
};
```
