# [989. Add to Array-Form of Integer](https://leetcode.com/problems/add-to-array-form-of-integer)

## Description

The **array-form** of an integer `num` is an array representing its digits in left to right order.

- For example, for `num = 1321`, the array form is `[1,3,2,1]`.

Given `num`, the **array-form** of an integer, and an integer `k`, return _the **array-form** of the integer_ `num + k`.

**Example 1:**

**Input:** num = \[1,2,0,0\], k = 34  
**Output:** \[1,2,3,4\]  
**Explanation:** 1200 + 34 = 1234

**Example 2:**

**Input:** num = \[2,7,4\], k = 181  
**Output:** \[4,5,5\]  
**Explanation:** 274 + 181 = 455

**Example 3:**

**Input:** num = \[2,1,5\], k = 806  
**Output:** \[1,0,2,1\]  
**Explanation:** 215 + 806 = 1021

**Constraints:**

- `1 <= num.length <= 10^4`
- `0 <= num[i] <= 9`
- `num` does not contain any leading zeros except for the zero itself.
- `1 <= k <= 10^4`

## My Solution

**JavaScript**

```js
const addToArrayForm = (num, k) => {
  const result = [];

  // Process from least significant digit
  for (let i = num.length - 1; i >= 0; i--) {
    const sum = num[i] + k;
    result.push(sum % 10);
    k = Math.floor(sum / 10);
  }

  // Remaining carry digits — also pushed in reverse
  while (k > 0) {
    result.push(k % 10);
    k = Math.floor(k / 10);
  }

  // One reverse at the end — O(n)
  result.reverse();

  return result;
};
```

```js
// Approach: Starting from right-most index, add numbers and carry over
// any 1's digits
const addToArrayForm = (num, k) => {
  const result = [...num];

  // Starting from last index, add the last digit of k to that element
  // Then reduce k by removing the digit that was added above
  for (let i = num.length - 1; i >= 0; --i) {
    result[i] = (num[i] + k) % 10;
    k = Math.floor((num[i] + k) / 10);
  }

  // While k is not zero, there are digits left to carry over
  while (k) {
    // Add those digits to the front of the list
    result.unshift(k % 10);
    k = Math.floor(k / 10);
  }

  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N + Log(K))   | O(N)             |

**Python**

def addToArrayForm(self, num: List[int], k: int) -> List[int]:
    result = []

    carry = k
    for i in range(len(num) - 1, -1, -1):
        carry += num[i]
        result.append(carry % 10)
        carry //= 10

    while carry:
        result.append(carry % 10)
        carry //= 10

    result.reverse()
    return result

**Java**

```java
public List<Integer> addToArrayForm(int[] num, int k) {
    List<Integer> result = new ArrayList<>();

    int carry = k;
    for (int i = num.length - 1; i >= 0; i--) {
        carry += num[i];
        result.add(carry % 10);
        carry /= 10;
    }

    while (carry > 0) {
        result.add(carry % 10);
        carry /= 10;
    }

    // One reverse — very fast in practice
    Collections.reverse(result);

    return result;
}
```
