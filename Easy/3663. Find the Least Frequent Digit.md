# [3663. Find the Least Frequent Digit](https://leetcode.com/problems/find-the-least-frequent-digit)

## Description

Given an integer `n`, find the digit that occurs **least** frequently in its decimal representation. If multiple digits have the same frequency, choose the **smallest** digit.

Return the chosen digit as an integer.

The **frequency** of a digit `x` is the number of times it appears in the decimal representation of `n`.

**Example 1:**

**Input:** n = 1553322  
**Output:** 1  
**Explanation:**  
The least frequent digit in `n` is 1, which appears only once. All other digits appear twice.

**Example 2:**

**Input:** n = 723344511  
**Output:** 2  
**Explanation:**  
The least frequent digits in `n` are 7, 2, and 5; each appears only once.

**Constraints:**

- `1 <= n <= 2^31​​​​​​​ - 1`

## My Solution

**JavaScript**

```js
const getLeastFrequentDigit = (n) => {
  let result = 0;
  let leastFrequentCount = 9;
  const digitCounts = new Array(10).fill(0);

  while (n) {
    const digit = n % 10;
    ++digitCounts[digit];
    n = Math.floor(n / 10);
  }

  for (let i = 0; i <= 9; ++i) {
    if (digitCounts[i] && digitCounts[i] < leastFrequentCount) {
      leastFrequentCount = digitCounts[i];
      result = i;
    }
  }

  return result;
};
```

| O(Log(N))       | O(1)             |

```js
// Approach: Using strings
const getLeastFrequentDigit = (n) => {
  const numStr = `${n}`;
  let leastFrequentCount = 9;
  let result = 0;

  for (let i = 0; i <= 9; ++i) {
    const regex = new RegExp(i, 'g');
    const digitCount = (numStr.match(regex) || []).length;

    if (digitCount !== 0 && digitCount < leastFrequentCount) {
      leastFrequentCount = digitCount;
      result = i;
    }
  }

  return result;
};
```
