# [3345. Smallest Divisible Digit Product I](https://leetcode.com/problems/)

## Description

You are given two integers `n` and `t`. Return the **smallest** number greater than or equal to `n` such that the **product of its digits** is divisible by `t`.

**Example 1:**

**Input:** n = 10, t = 2  
**Output:** 10  
**Explanation:**  
The digit product of 10 is 0, which is divisible by 2, making it the smallest number greater than or equal to 10 that satisfies the condition.

**Example 2:**

**Input:** n = 15, t = 3  
**Output:** 16  
**Explanation:**  
The digit product of 16 is 6, which is divisible by 3, making it the smallest number greater than or equal to 15 that satisfies the condition.

**Constraints:**

- `1 <= n <= 100`
- `1 <= t <= 10`

## My Solution

**JavaScript**

```js
// We only have to check up to 10 numbers, as any number with
// a 0 in it is divisible by any number t.
const smallestNumber = (n, t) => {
  for (let i = n; i < n + 10; ++i) {
    if (digitProduct(i) % t === 0) return i;
  }
};

const digitProduct = (n) => {
  result = 1;

  while (n) {
    const digit = n % 10;
    if (digit === 0) return 0;
    result *= digit;
    n = Math.floor(n / 10);
  }

  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(Log(N))       | O(N)             |

**Python**

```python
def smallestNumber(self, n: int, t: int) -> int:
    def digitProduct(n):
        result = 1

        while n:
            digit = n % 10
            result *= digit
            n //= 10

        return result

    for i in range(n, n + 10):
        if digitProduct(i) % t == 0:
            return i
```

| --------------- | ---------------- |
