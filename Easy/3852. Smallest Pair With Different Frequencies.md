# [3852. Smallest Pair With Different Frequencies](https://leetcode.com/problems/smallest-pair-with-different-frequencies)

## Description

You are given an integer array `nums`.

Consider all pairs of **distinct** values `x` and `y` from `nums` such that:

- `x < y`
- `x` and `y` have different frequencies in `nums`.

Among all such pairs:

- Choose the pair with the smallest possible value of `x`.
- If multiple pairs have the same `x`, choose the one with the smallest possible value of `y`.

Return an integer array `[x, y]`. If no valid pair exists, return `[-1, -1]`.

**Example 1:**

**Input:** nums = \[1,1,2,2,3,4\]  
**Output:** \[1,3\]  
**Explanation:**

The smallest value is 1 with a frequency of 2, and the smallest value greater than 1 that has a different frequency from 1 is 3 with a frequency of 1. Thus, the answer is `[1, 3]`.

**Example 2:**

**Input:** nums = \[1,5\]  
**Output:** \[-1,-1\]  
**Explanation:**  
Both values have the same frequency, so no valid pair exists. Return `[-1, -1]`.

**Example 3:**

**Input:** nums = \[7\]  
**Output:** \[-1,-1\]  
**Explanation:**  
There is only one value in the array, so no valid pair exists. Return `[-1, -1]`.

**Constraints:**

- `1 <= nums.length <= 100`
- `1 <= nums[i] <= 100`

## My Solution

**JavaScript**

```js
// Avoid sorting by using an array as a bucket, where index 0 stores
// the frequency of 0's, index 1 the frequency of 1's, etc.
const minDistinctFreqPair = (nums) => {
  const counts = new Array(101).fill(0);
  for (const num of nums) counts[num]++;

  let firstNum = -1;

  // The index corresponds to the number being examined
  for (let curNum = 0; curNum < counts.length; ++curNum) {
    // Disregard numbers that aren't even part of input array
    if (counts[curNum] <= 0) continue;

    // Lock in the first valid number we see
    if (firstNum === -1) firstNum = curNum;
    // If we have a first number locked in, check subsequence numbers for differing counts
    else if (counts[curNum] !== counts[firstNum]) return [firstNum, curNum];
  }

  // No valid pair was found
  return [-1, -1];
};
```

```python
def minDistinctFreqPair(self, nums: list[int]) -> list[int]:
    counts = [0] * 101
    for num in nums: counts[num] += 1

    first_num = -1

    for cur_num in range(len(counts)):
        if counts[cur_num] <= 0: continue
        if first_num == -1: first_num = cur_num
        elif counts[cur_num] != counts[first_num]: return [first_num, cur_num]

    return [-1, -1]
```

```java
public int[] minDistinctFreqPair(int[] nums) {
    int[] counts = new int[101];
    for (int num : nums) counts[num]++;

    int firstNum = -1;

    for (int curNum = 0; curNum < counts.length; ++curNum) {
        if (counts[curNum] <= 0) continue;
        if (firstNum == -1) firstNum = curNum;
        else if (counts[curNum] != counts[firstNum]) {
            return new int[]{firstNum, curNum};
        }

    return new int[]{-1, -1};
}
```
