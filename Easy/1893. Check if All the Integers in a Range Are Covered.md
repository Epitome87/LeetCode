# [1893. Check if All the Integers in a Range Are Covered](https://leetcode.com/problems/check-if-all-the-integers-in-a-range-are-covered)

## Description

You are given a 2D integer array `ranges` and two integers `left` and `right`. Each `ranges[i] = [starti, endi]` represents an **inclusive** interval between `starti` and `endi`.

Return `true` _if each integer in the inclusive range_ `[left, right]` _is covered by **at least one** interval in_ `ranges`. Return `false` _otherwise_.

An integer `x` is covered by an interval `ranges[i] = [starti, endi]` if `starti <= x <= endi`.

**Example 1:**

**Input:** ranges = \[\[1,2\],\[3,4\],\[5,6\]\], left = 2, right = 5  
**Output:** true  
**Explanation:** Every integer between 2 and 5 is covered:

- 2 is covered by the first range.
- 3 and 4 are covered by the second range.
- 5 is covered by the third range.

**Example 2:**

**Input:** ranges = \[\[1,10\],\[10,20\]\], left = 21, right = 21  
**Output:** false  
**Explanation:** 21 is not covered by any range.

**Constraints:**

- `1 <= ranges.length <= 50`
- `1 <= starti <= endi <= 50`
- `1 <= left <= right <= 50`

## My Solution

**JavaScript**

```js
const isCovered = (ranges, left, right) => {
  // Timeline array. Note the max value can be 50, so size 52 lets us mark an event at 51 (end + 1)
  const diff = new Array(52).fill(0);

  for (const [start, end] of ranges) {
    // A new interval starts providing coverage at this point
    diff[start]++;
    // That coverage drops off AFTER the end point (end is inclusive)
    diff[end + 1]--;
  }

  // Sweep the timeline
  let overlaps = 0;
  for (let i = 1; i <= right; ++i) {
    // Accumulate events sitting at this exact point
    overlaps += diff[i];

    // Are we currently inside our target window, but the coverage is 0?
    // If so, we have a lapse in coverage
    if (i >= left && overlaps === 0) return false;
  }

  return true;
};
```

| --------------- | ---------------- |

def isCovered(self, ranges: List[List[int]], left: int, right: int) -> bool:
    timeline = [0] * 52

    for start, end in ranges:
        timeline[start] += 1
        timeline[end + 1] -= 1

    overlaps = 0
    for i in range(1, right + 1):
        overlaps += timeline[i]

        if i >= left and overlaps == 0:
            return False

    return True

public boolean isCovered(int[][] ranges, int left, int right) {
    int[] timeline = new int[52];

    for (int[] range : ranges) {
        int start = range[0];
        int end = range[1];
        timeline[start]++;
        timeline[end + 1]--;
    }

    int overlaps = 0;
    for (int i = 1; i <= right; ++i) {
        overlaps += timeline[i];

        if (i >= left && overlaps == 0) return false;
    }

    return true;
}
