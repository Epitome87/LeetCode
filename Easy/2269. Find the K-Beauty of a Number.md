# [2269. Find the K-Beauty of a Number](https://leetcode.com/problems/find-the-k-beauty-of-a-number)

## Description

The **k-beauty** of an integer `num` is defined as the number of **substrings** of `num` when it is read as a string that meet the following conditions:

- It has a length of `k`.
- It is a divisor of `num`.

Given integers `num` and `k`, return _the k-beauty of_ `num`.

Note:

- **Leading zeros** are allowed.
- `0` is not a divisor of any value.

A **substring** is a contiguous sequence of characters in a string.

**Example 1:**

**Input:** num = 240, k = 2  
**Output:** 2  
**Explanation:** The following are the substrings of num of length k:

- "24" from "**24**0": 24 is a divisor of 240.
- "40" from "2**40**": 40 is a divisor of 240.  
  Therefore, the k-beauty is 2.

**Example 2:**

**Input:** num = 430043, k = 2  
**Output:** 2  
**Explanation:** The following are the substrings of num of length k:

- "43" from "**43**0043": 43 is a divisor of 430043.
- "30" from "4**30**043": 30 is not a divisor of 430043.
- "00" from "43**00**43": 0 is not a divisor of 430043.
- "04" from "430**04**3": 4 is not a divisor of 430043.
- "43" from "4300**43**": 43 is a divisor of 430043.  
  Therefore, the k-beauty is 2.

**Constraints:**

- `1 <= num <= 10^9`
- `1 <= k <= num.length` (taking `num` as a string)

## My Solution

**JavaScript**

```js
// Approach: Avoiding using string conversions.
const divisorSubstrings = (num, k) => {
  let result = 0;
  let originalNum = num;
  let power = Math.pow(10, k);
  let divisor = 0;

  // While we still have at least k digits...
  // We will extract k-digit numbers from the end and move left
  while (num >= Math.pow(10, k - 1)) {
    // Get the last k digits
    divisor = num % power;

    // Do we have a valid result?
    if (divisor !== 0 && originalNum % divisor === 0) ++result;

    // Remove the last digit (slide window to left)
    num = Math.floor(num / 10);
  }

  return result;
};
```

| --------------- | ---------------- |
| O(Log(N))       | O(1)             |

```js
// Approach: Brute-Force using string conversion
const divisorSubstrings = (num, k) => {
  let result = 0;
  const numStr = num.toString();

  for (let i = 0; i <= numStr.length - k; ++i) {
    const substr = numStr.slice(i, i + k);
    if (num % parseInt(substr) === 0) {
      ++result;
    }
  }

  return result;
};
```

| Time Complexity                         | Space Complexity |
| --------------------------------------- | ---------------- |
| O(N \* K, K = length of each substring) | O(N)             |

**Python**

```python
def divisorSubstrings(self, num: int, k: int) -> int:
    result = 0
    original_num = num
    power = 10 ** k

    while num >= 10 ** (k - 1):
        # Extract the last k digits
        divisor = num % power

        if divisor != 0 and original_num % divisor == 0:
            result += 1

        # Remove rightmost digit from num. This slides our window over
        # Next iteration, divisor will grab the next set of k digits
        num = num // 10

    return result
```

public int divisorSubstrings(int num, int k) {
    int result = 0;
    int originalNum = num;
    int power = (int) Math.pow(10, k);

    while (num >= Math.pow(10, k - 1)) {
        int divisor = num % power;

        if (divisor != 0 && originalNum % divisor == 0) {
            ++result;
        }

        num = num / 10;
    }

    return result;
}
