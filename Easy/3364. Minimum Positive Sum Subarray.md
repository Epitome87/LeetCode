# [3364. Minimum Positive Sum Subarray](https://leetcode.com/problems/minimum-positive-sum-subarray)

## Description

You are given an integer array `nums` and **two** integers `l` and `r`. Your task is to find the **minimum** sum of a **subarray** whose size is between `l` and `r` (inclusive) and whose sum is greater than 0.

Return the **minimum** sum of such a subarray. If no such subarray exists, return -1.

A **subarray** is a contiguous **non-empty** sequence of elements within an array.

**Example 1:**

**Input:** nums = \[3, -2, 1, 4\], l = 2, r = 3  
**Output:** 1  
**Explanation:**  
The subarrays of length between `l = 2` and `r = 3` where the sum is greater than 0 are:

- `[3, -2]` with a sum of 1
- `[1, 4]` with a sum of 5
- `[3, -2, 1]` with a sum of 2
- `[-2, 1, 4]` with a sum of 3

Out of these, the subarray `[3, -2]` has a sum of 1, which is the smallest positive sum. Hence, the answer is 1.

**Example 2:**

**Input:** nums = \[-2, 2, -3, 1\], l = 2, r = 3  
**Output:** \-1  
**Explanation:**  
There is no subarray of length between `l` and `r` that has a sum greater than 0. So, the answer is -1.

**Example 3:**

**Input:** nums = \[1, 2, 3, 4\], l = 2, r = 4  
**Output:** 3  
**Explanation:**  
The subarray `[1, 2]` has a length of 2 and the minimum sum greater than 0. So, the answer is 3.

**Constraints:**

- `1 <= nums.length <= 100`
- `1 <= l <= r <= nums.length`
- `-1000 <= nums[i] <= 1000`

## My Solution

**JavaScript**

```js
const minimumSumSubarray = (nums, l, r) => {
  let result = Infinity;

  // Iterate the window size, starting at l and ending at r
  for (let windowSize = l; windowSize <= r; ++windowSize) {
    // Calculate initial window sum
    let windowSum = 0;
    for (let i = 0; i < windowSize; ++i) windowSum += nums[i];

    // Is the initial window sum a potential result?
    if (windowSum > 0) result = Math.min(result, windowSum);

    // Slide the window to find the sum of all subarrays of size windowSize
    for (let i = windowSize; i < nums.length; ++i) {
      // Add contribution of i, subtract contribution of i - windowSize
      windowSum += nums[i] - nums[i - windowSize];

      // Did we find a valid, potential result?
      if (windowSum > 0) result = Math.min(result, windowSum);
    }
  }

  return result === Infinity ? -1 : result;
};
```

| ------------------- | ---------------- |
| O((R - L + 1) \* N) | O(1)             |

def minimumSumSubarray(self, nums: List[int], l: int, r: int) -> int:
    result = float('inf')

    for window_size in range(l, r + 1):
        window_sum = 0
        for i in range(window_size):
            window_sum += nums[i]

        if window_sum > 0:
            result = min(result, window_sum)

        for i in range(window_size, len(nums)):
            window_sum += nums[i] - nums[i - window_size]

            if window_sum > 0: result = min(result, window_sum)

    return -1 if result == float('inf') else result

**Java**

```java
public int minimumSumSubarray(List<Integer> nums, int l, int r) {
    int result = Integer.MAX_VALUE;

    for (int windowSize = l; windowSize <= r; ++windowSize) {
        int sum = 0;

        // Slide the window
        for (int i = 0; i < nums.size(); ++i) {
            sum += nums.get(i);
            if (i >= windowSize - 1) {
                if (sum > 0) result = Math.min(result, sum);
                sum -= nums.get(i - windowSize + 1);
            }
        }
    }

    return result == Integer.MAX_VALUE ? -1 : result;
}
```
