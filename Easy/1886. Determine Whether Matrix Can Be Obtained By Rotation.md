# [1886. Determine Whether Matrix Can Be Obtained By Rotation](https://leetcode.com/problems/determine-whether-matrix-can-be-obtained-by-rotation)

## Description

Given two `n x n` binary matrices `mat` and `target`, return `true` _if it is possible to make_ `mat` _equal to_ `target` _by **rotating**_ `mat` _in **90-degree increments**, or_ `false` _otherwise._

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/05/20/grid3.png)

**Input:** mat = \[\[0,1\],\[1,0\]\], target = \[\[1,0\],\[0,1\]\]  
**Output:** true  
**Explanation:** We can rotate mat 90 degrees clockwise to make mat equal target.

**Example 2:**

![](https://assets.leetcode.com/uploads/2021/05/20/grid4.png)

**Input:** mat = \[\[0,1\],\[1,1\]\], target = \[\[1,0\],\[0,1\]\]  
**Output:** false  
**Explanation:** It is impossible to make mat equal to target by rotating mat.

**Example 3:**

![](https://assets.leetcode.com/uploads/2021/05/26/grid4.png)

**Input:** mat = \[\[0,0,0\],\[0,1,0\],\[1,1,1\]\], target = \[\[1,1,1\],\[0,1,0\],\[0,0,0\]\]  
**Output:** true  
**Explanation:** We can rotate mat 90 degrees clockwise two times to make mat equal target.

**Constraints:**

- `n == mat.length == target.length`
- `n == mat[i].length == target[i].length`
- `1 <= n <= 10`
- `mat[i][j]` and `target[i][j]` are either `0` or `1`.

```js
const findRotation = (mat, target) => {
  for (let rot = 0; rot < 360; rot += 90) {
    if (isEqual(mat, target, rot)) return true;
  }

  return false;
};

const isEqual = (mat, target, rotation) => {
  const n = mat[0].length;

  switch (rotation) {
    case 0:
      for (let r = 0; r < n; ++r) {
        for (let c = 0; c < n; ++c) {
          if (mat[r][c] !== target[r][c]) {
            return false;
          }
        }
      }
      return true;

    case 90:
      for (let r = 0; r < n; ++r) {
        for (let c = 0; c < n; ++c) {
          if (mat[r][c] !== target[c][n - r - 1]) {
            return false;
          }
        }
      }
      return true;

    case 180:
      for (let r = 0; r < n; ++r) {
        for (let c = 0; c < n; ++c) {
          if (mat[r][c] !== target[n - r - 1][n - c - 1]) {
            return false;
          }
        }
      }
      return true;

    case 270:
      for (let r = 0; r < n; ++r) {
        for (let c = 0; c < n; ++c) {
          if (mat[r][c] !== target[n - c - 1][r]) {
            return false;
          }
        }
      }
      return true;
  }
};
```

**Python**

```python
def findRotation(self, mat: List[List[int]], target: List[List[int]]) -> bool:
  n = len(mat)

  # Flags for 0, 90, 180, and 270 degrees
  valid_rotations = [True, True, True, True]

  for r in range(n):
      for c in range(n):
          # 0 degrees
          if valid_rotations[0] and mat[r][c] != target[r][c]:
              valid_rotations[0] = False

          # 90 degrees
          if valid_rotations[1] and mat[r][c] != target[c][n - r - 1]:
              valid_rotations[1] = False

          # 180 degrees
          if valid_rotations[2] and mat[r][c] != target[n - r - 1][n - c - 1]:
              valid_rotations[2] = False

          # 270 degrees
          if valid_rotations[3] and mat[r][c] != target[n - c - 1][r]:
              valid_rotations[3] = False

          # Return early if all 4 rotations are found to be invalid
          if not any(valid_rotations):
              return False

  # If we finished the loop and at least one is still True, a rotation exists
  return True
```

**Java**

```java
public boolean findRotation(int[][] mat, int[][] target) {
    int n = mat.length;
    boolean[] validRotations = {true, true, true, true};

    for (int r = 0; r < n; r++) {
        for (int c = 0; c < n; c++) {
            // 0 degrees
            if (validRotations[0] && mat[r][c] != target[r][c]) {
                validRotations[0] = false;
            }

            // 90 degrees
            if (validRotations[1] && mat[r][c] != target[c][n - r - 1]) {
                validRotations[1] = false;
            }

            // 180 degrees
            if (validRotations[2] && mat[r][c] != target[n - r - 1][n - c - 1]) {
                validRotations[2] = false;
            }

            // 270 degrees
            if (validRotations[3] && mat[r][c] != target[n - c - 1][r]) {
                validRotations[3] = false;
            }

            if (!validRotations[0] && !validRotations[1] &&
                !validRotations[2] && !validRotations[3]) {
                return false;
            }
        }
    }

    return true;
}
```
