# [37. Sudoku Solver](https://leetcode.com/problems/sudoku-solver)

## Description

Write a program to solve a Sudoku puzzle by filling the empty cells.

A sudoku solution must satisfy **all of the following rules**:

1.  Each of the digits `1-9` must occur exactly once in each row.
2.  Each of the digits `1-9` must occur exactly once in each column.
3.  Each of the digits `1-9` must occur exactly once in each of the 9 `3x3` sub-boxes of the grid.

The `'.'` character indicates empty cells.

**Constraints:**

- `board.length == 9`
- `board[i].length == 9`
- `board[i][j]` is a digit or `'.'`.
- It is **guaranteed** that the input board has only one solution.

## My Solution

**JavaScript**

```js
const solveSudoku = (board) => {
  const ROWS = board.length;
  const COLS = board[0].length;

  const isValid = (r, c, num) => {
    const startRow = Math.floor(r / 3) * 3;
    const startCol = Math.floor(c / 3) * 3;

    for (let i = 0; i < COLS; ++i) {}
  };

  const solve = (r, c) => {
    // If we've moved past final row, we've filled out entire board
    if (r === ROWS) return true;
    // If we've reached past current column, wrap to next row
    if (c === COLS) return solve(r + 1, 0);
    // If the current cell is already filled, move to next column
    if (board[r][c] !== '.') return solve(r, c + 1);

    for (let i = 1; i <= 9; ++i) {
      const num = i.toString();

      if (isValidRow(r, num) && isValidCol(c, num) && isValidSub(r, c, num)) {
        // Choose this number for this row/col
        board[r][c] = num;
        // Explore (move to next column)
        if (solve(r, c + 1)) return true;
        // Unchoose (backtrack, as this number didn't lead to solving puzzle)
        board[r][c] = '.';
      }
    }

    // Numbers 1-9 all failed; this path is dead! Trigger backtrack
    return false;
  };

  solve(0, 0);
  return board;
};
```
