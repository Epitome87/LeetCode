# [23. Merge K Sorted Lists](https://leetcode.com/problems/merge-k-sorted-lists)

## Description

You are given an array of `k` linked-lists `lists`, each linked-list is sorted in ascending order.

_Merge all the linked-lists into one sorted linked-list and return it._

**Example 1:**

**Input:** lists = \[\[1,4,5\],\[1,3,4\],\[2,6\]\]  
**Output:** \[1,1,2,3,4,4,5,6\]  
**Explanation:** The linked-lists are:  
\[  
 1->4->5,  
 1->3->4,  
 2->6  
\]  
merging them into one sorted linked list:  
1->1->2->3->4->4->5->6

**Example 2:**

**Input:** lists = \[\]  
**Output:** \[\]

**Example 3:**

**Input:** lists = \[\[\]\]  
**Output:** \[\]

**Constraints:**

- `k == lists.length`
- `0 <= k <= 10^4`
- `0 <= lists[i].length <= 500`
- `-10^4 <= lists[i][j] <= 10^4`
- `lists[i]` is sorted in **ascending order**.
- The sum of `lists[i].length` will not exceed `10^4`.

## My Solution

**JavaScript**

```js
const mergeKLists = (lists) => {
  if (!lists || lists.length === 0) return null;

  /* While there are at least two linked lists needing merged, we will merge two pairs
  at a time in rounds. This allows us to merge lists while they are at a smaller size
  For example, if there are 6 lists, round 1 will merge List 1 and 2, List 3 and 4, and
  List 5 and 6. Next round begins with only the 3 resulting merged lists of the previous round,
  and thus pairs the resulting L1 and L2, as well as L3 with null (as there is no even-number pairing for it).
  In the third round, the result of L1/L2 merge is merged with the result of the L3/null merge, resulting in
  one final, length-1 fully-merged list. */
  while (lists.length > 1) {
    // Temp array holds the result of the current round of 2-list pairings
    const temp = [];

    // Step by i += 2 so we can grab lists in pairs (List 0 and 1, 2 and 3, etc)
    for (let i = 0; i < lists.length; i += 2) {
      const l1 = lists[i];

      // If we have an odd number of lists, the last list won't have a pairing
      const l2 = i + 1 < lists.length ? lists[i + 1] : null;

      // Merge the pair together, pushing the resulting merged list into temp
      temp.push(mergeLists(l1, l2));
    }

    // Prepare for next round of iteration by overwriting input array with
    // newly merged lists. This effectively cuts the lists size in half each time
    lists = temp;
  }

  // At this point, all nodes have been merged into a single list which
  // resides as the only element remaining in the lists array
  return lists[0];
};

// Helper function to merge two Linked Lists
const mergeLists = (l1, l2) => {
  let node = new ListNode();
  const dummy = node;

  while (l1 && l2) {
    if (l1.val <= l2.val) {
      node.next = l1;
      l1 = l1.next;
    } else {
      node.next = l2;
      l2 = l2.next;
    }
    node = node.next;
  }

  node.next = l1 || l2;
  return dummy.next;
};
```

**Java**

```java
public ListNode mergeKLists(ListNode[] lists) {
    if (lists == null || lists.length == 0) return null;

    while (lists.length > 1) {
        ListNode[] mergedLists = new ListNode[(lists.length + 1) / 2];
        int index = 0;

        for (int i = 0; i < lists.length; i += 2) {
            ListNode list1 = lists[i];
            ListNode list2 = (i + 1 < lists.length) ? lists[i + 1] : null;
            mergedLists[index++] = mergeTwoLists(list1, list2);
        }

        lists = mergedLists;
    }

    return lists[0];
}

private ListNode mergeTwoLists(ListNode list1, ListNode list2) {
    ListNode dummy = new ListNode(0);
    ListNode current = dummy;

    while (list1 != null && list2 != null) {
        if (list1.val <= list2.val) {
            current.next = list1;
            list1 = list1.next;
        } else {
            current.next = list2;
            list2 = list2.next;
        }
        current = current.next;
    }

    if (list1 != null) current.next = list1;
    else if (list2 != null) current.next = list2;

    return dummy.next;
}
```
