# [198. House Robber](https://leetcode.com/problems/house-robber)

## Description

You are a professional robber planning to rob houses along a street. Each house has a certain amount of money stashed, the only constraint stopping you from robbing each of them is that adjacent houses have security systems connected and **it will automatically contact the police if two adjacent houses were broken into on the same night**.

Given an integer array `nums` representing the amount of money of each house, return _the maximum amount of money you can rob tonight **without alerting the police**_.

**Example 1:**

**Input:** nums = \[1,2,3,1\]  
**Output:** 4  
**Explanation:** Rob house 1 (money = 1) and then rob house 3 (money = 3).  
Total amount you can rob = 1 + 3 = 4.

**Example 2:**

**Input:** nums = \[2,7,9,3,1\]  
**Output:** 12  
**Explanation:** Rob house 1 (money = 2), rob house 3 (money = 9) and rob house 5 (money = 1).  
Total amount you can rob = 2 + 9 + 1 = 12.

**Constraints:**

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 400`

## My Solution

**JavaScript**

```js
// Approach: Dynamic Programming, space-optimized
const rob = (nums) => {
  const n = nums.length;
  if (n === 1) return nums[0];

  let previous2 = 0;
  let previous1 = 0;

  for (const num of nums) {
    const temp = previous1;
    previous1 = Math.max(previous2 + num, previous1);
    previous2 = temp;
  }

  return previous1;
};
```

```js
// Approach: Dynamic Programming, Bottom-Up
const rob = (nums) => {
  const n = nums.length;
  if (n === 1) return nums[0];

  const dp = Array(n).fill(0);
  dp[0] = nums[0];
  dp[1] = Math.max(nums[0], nums[1]);

  for (let i = 2; i < n; ++i) {
    // Do we rob the previous house, or this house and the one before the last?
    dp[i] = Math.max(dp[i - 1], nums[i] + dp[i - 2]);
  }

  return dp.at(-1);
};
```

```python
def rob(self, nums: List[int]) -> int:
    n = len(nums)
    if n == 1: return nums[0]

    # Initial the DP array with all 0s
    dp = [0] * n
    dp[0] = nums[0] # No better option at first house
    dp[1] = max(nums[0], nums[1]) # Is house 1 or house 2 better?

    # For house 3 onward, is robbing the current house and 2nd to last
    # more lucrative than robbing the previous house?
    for i in range(2, n):
        dp[i] = max(dp[i - 2] + nums[i], dp[i - 1])

    return dp[-1]
```

| O(N)            | O(N)             |
