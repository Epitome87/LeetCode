# [784. Letter Case Permutation](https://leetcode.com/problems/letter-case-permutation)

## Description

Given a string `s`, you can transform every letter individually to be lowercase or uppercase to create another string.

Return _a list of all possible strings we could create_. Return the output in **any order**.

**Example 1:**

**Input:** s = "a1b2"  
**Output:** \["a1b2","a1B2","A1b2","A1B2"\]

**Example 2:**

**Input:** s = "3z4"  
**Output:** \["3z4","3Z4"\]

**Constraints:**

- `1 <= s.length <= 12`
- `s` consists of lowercase English letters, uppercase English letters, and digits.

## My Solution

**JavaScript**

```js
const letterCasePermutation = (s) => {
  const result = [];

  const backtrack = (prefix, idx) => {
    if (idx === s.length) {
      result.push(prefix);
      return;
    }

    if (/[a-z]/gi.test(s[idx])) {
      // Choose lowercase
      backtrack(prefix + s[idx].toLowerCase(), idx + 1);

      // Chooser uppercase
      backtrack(prefix + s[idx].toUpperCase(), idx + 1);
    } else {
      // Not a letter; only chose this character
      backtrack(prefix + s[idx], idx + 1);
    }
  };

  backtrack('', 0);
  return result;
};
```

| Time Complexity                    | Space Complexity |
| ---------------------------------- | ---------------- |
| O(N \* 2^K, K = number of letters) | O(N \* 2^K)      |

**Python**

```python
def letterCasePermutation(self, s: str) -> List[str]:
    result = []

    def backtrack(prefix: str, idx: int):
        if idx == len(s):
            result.append(prefix)
            return

        if s[idx].isalpha():
            backtrack(prefix + s[idx].lower(), idx + 1)
            backtrack(prefix + s[idx].upper(), idx + 1)
        else:
            backtrack(prefix + s[idx], idx + 1)

    backtrack('', 0)
    return result
```

| Time Complexity                    | Space Complexity |
| ---------------------------------- | ---------------- |
| O(N \* 2^K, K = number of letters) | O(N \* 2^K)      |
