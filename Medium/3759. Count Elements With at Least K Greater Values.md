# [3759. Count Elements With at Least K Greater Values](https://leetcode.com/problems/count-elements-with-at-least-k-greater-values)

## Description

You are given an integer array `nums` of length `n` and an integer `k`.

An element in `nums` is said to be **qualified** if there exist **at least** `k` elements in the array that are **strictly greater** than it.

Return an integer denoting the total number of qualified elements in `nums`.

**Example 1:**

**Input:** nums = \[3,1,2\], k = 1  
**Output:** 2  
**Explanation:**  
The elements 1 and 2 each have at least `k = 1` element greater than themselves.  
​​​​​​​No element is greater than 3. Therefore, the answer is 2.

**Example 2:**

**Input:** nums = \[5,5,5\], k = 2  
**Output:** 0  
**Explanation:**  
Since all elements are equal to 5, no element is greater than the other. Therefore, the answer is 0.

**Constraints:**

- `1 <= n == nums.length <= 10^5`
- `1 <= nums[i] <= 10^9`
- `0 <= k < n`

## My Solution

**JavaScript**

```js
const countElements = (nums, k) => {
  if (k === 0) return nums.length;

  const sortedNums = nums.toSorted((a, b) => a - b);
  const threshold = sortedNums[nums.length - k];

  // Use binary search to find first index where num >= threshold
  let left = 0;
  let right = sortedNums.length;

  while (left < right) {
    const mid = (left + right) >> 1;
    if (sortedNums[mid] < threshold) left = mid + 1;
    else right = mid;
  }

  return left;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N \* Log(N))  | O(Sorting)       |

```js
const countElements = (nums, k) => {

  const sortedNums = nums.toSorted((a, b) => a - b);
  const threshold = sortedNums[nums.length - k];
```

**Python**

```python
def countElements(self, nums: List[int], k: int) -> int:
    if k == 0: return len(nums)

    result = 0
    sorted_nums = sorted(nums)
    threshold = sorted_nums[len(nums) - k]

    for num in sorted_nums:
        if num < threshold: result += 1
        else: break

    return result
```

**Java**

```java
public int countElements(int[] nums, int k) {
    if (k == 0) return nums.length;

    int result = 0;
    int[] sortedNums = nums.clone();
    Arrays.sort(sortedNums);
    int threshold = sortedNums[nums.length - k];

    for (int num : sortedNums) {
        if (num < threshold) ++result;
        else break;
    }

    return result;
}
```
