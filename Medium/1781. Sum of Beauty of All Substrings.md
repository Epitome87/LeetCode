# [1781. Sum of Beauty of All Substrings](https://leetcode.com/problems/sum-of-beauty-of-all-substrings)

## Description

The **beauty** of a string is the difference in frequencies between the most frequent and least frequent characters.

- For example, the beauty of `"abaacc"` is `3 - 1 = 2`.

Given a string `s`, return _the sum of **beauty** of all of its substrings._

**Example 1:**

**Input:** s = "aabcb"  
**Output:** 5  
**Explanation:** The substrings with non-zero beauty are \["aab","aabc","aabcb","abcb","bcb"\], each with beauty equal to 1.

**Example 2:**

**Input:** s = "aabcbaa"  
**Output:** 17

**Constraints:**

- `1 <= s.length <= 500`
- `s` consists of only lowercase English letters.

## My Solution

**JavaScript**

```js
// Approach: O(N^2) with constant space
const beautySum = (s) => {
  const n = s.length;
  let beautySum = 0;

  for (let i = 0; i < n; ++i) {
    const freq = new Array(26).fill(0);

    for (let j = i; j < n; ++j) {
      freq[s.charCodeAt(j) - 97]++;

      let min = Infinity;
      let max = -Infinity;

      for (let char = 0; char < 26; ++char) {
        const charFreq = freq[char];
        if (charFreq > 0) {
          min = Math.min(min, charFreq);
          max = Math.max(max, charFreq);
        }
      }

      beautySum += max - min;
    }
  }

  return beautySum;
};
```

```js
// Approach: Brute-Force
const beautySum = (s) => {
  let beautySum = 0;
  const freqMap = new Map();

  for (let i = 0; i < s.length - 1; ++i) {
    // Reset frequency count, as we are rebuilding new substring
    freqMap.clear();

    for (let j = i; j < s.length; ++j) {
      // Increment frequency of each char processed
      freqMap.set(s[j], (freqMap.get(s[j]) || 0) + 1);

      // Add beauty of substring s[i...j]
      beautySum += beauty(freqMap);
    }
  }

  return beautySum;
};

const beauty = (freq) => {
  // We only care if multiple frequencies exist
  if (freq.size < 2) return 0;

  let min = Infinity;
  let max = -Infinity;

  for (const count of freq.values()) {
    if (count < min) min = count;
    if (count > max) max = count;
  }

  return max - min;
};
```

**Python**

```python
def beautySum(self, s: str) -> int:
    n = len(s)
    beauty_sum = 0

    for i in range(0, n):
        freq = [0] * 26

        for j in range(i, n):
            freq[ord(s[j]) - 97] += 1

            max_freq = max(freq)
            min_freq = min(x for x in freq if x > 0)
            beauty_sum += max_freq - min_freq

    return beauty_sum
```
