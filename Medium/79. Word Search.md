# [79. Word Search](https://leetcode.com/problems/word-search)

## Description

Given an `m x n` grid of characters `board` and a string `word`, return `true` _if_ `word` _exists in the grid_.

The word can be constructed from letters of sequentially adjacent cells, where adjacent cells are horizontally or vertically neighboring. The same letter cell may not be used more than once.

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/11/04/word2.jpg)

**Input:** board = \[\["A","B","C","E"\],\["S","F","C","S"\],\["A","D","E","E"\]\], word = "ABCCED"  
**Output:** true

**Example 2:**

![](https://assets.leetcode.com/uploads/2020/11/04/word-1.jpg)

**Input:** board = \[\["A","B","C","E"\],\["S","F","C","S"\],\["A","D","E","E"\]\], word = "SEE"  
**Output:** true

**Example 3:**

![](https://assets.leetcode.com/uploads/2020/10/15/word3.jpg)

**Input:** board = \[\["A","B","C","E"\],\["S","F","C","S"\],\["A","D","E","E"\]\], word = "ABCB"  
**Output:** false

**Constraints:**

- `m == board.length`
- `n = board[i].length`
- `1 <= m, n <= 6`
- `1 <= word.length <= 15`
- `board` and `word` consists of only lowercase and uppercase English letters.

**Follow up:** Could you use search pruning to make your solution faster with a larger `board`?

## My Solution

**JavaScript**

```js
const exist = (board, word) => {
  // Return early if the word is larger than total number of cells.
  if (word.length > board.length * board[0].length) return false;

  const rows = board.length;
  const cols = board[0].length;

  const backtrack = (row, col, idx) => {
    // If the index reaches word length, we discovered the full word!
    if (idx >= word.length) return true;

    // If index is out of bounds or the cell is not the current required character...
    if (row < 0 || row >= rows || col < 0 || col >= cols || board[row][col] !== word[idx]) {
      // This path didn't lead to the word
      return false;
    }

    // We need to temporarily mark this cell as visited, so we don't revisit it for this permutation
    board[row][col] = '0';

    // For each adjacent cell, check if ANY lead to the full word
    const directions = [
      [0, 1],
      [0, -1],
      [1, 0],
      [-1, 0],
    ];
    for (const [dr, dc] of directions) {
      if (backtrack(row + dr, col + dc, idx + 1)) return true;
    }

    // Restore the board to its original state
    board[row][col] = word[idx];

    return false;
  };

  // Begin the search for the start of the word at each possible cell.
  // If one of the cells returns true, it means that starting position
  // led to the discover of the full word!
  for (let row = 0; row < rows; ++row) {
    for (let col = 0; col < cols; ++col) {
      if (backtrack(row, col, 0)) return true;
    }
  }

  // We checked every cell on the board and none led to the word being found
  return false;
};
```

| Time Complexity                     | Space Complexity  |
| ----------------------------------- | ----------------- |
| O(Rows x Cols x 4^(Length of Word)) | O(Length of Word) |
