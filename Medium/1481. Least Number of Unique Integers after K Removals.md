# [1481. Least Number of Unique Integers after K Removals](https://leetcode.com/problems/least-number-of-unique-integers-after-k-removals)

## Description

Given an array of integers `arr` and an integer `k`. Find the *least number of unique integers* after removing **exactly** `k` elements**.**

**Example 1:**

**Input:** arr = \[5,5,4\], k = 1  
**Output:** 1  
**Explanation**: Remove the single 4, only 5 is left.

**Example 2:**

**Input:** arr = \[4,3,1,1,3,3,2\], k = 3  
**Output:** 2  
**Explanation**: Remove 4, 2 and either one of the two 1s or three 3s. 1 and 3 will be left.

**Constraints:**

- `1 <= arr.length <= 10^5`
- `1 <= arr[i] <= 10^9`
- `0 <= k <= arr.length`

## My Solution

**JavaScript**

```js
// Approach: Greedily prioritize removing numbers with the lowest frequencies.
// We can track the number of unique characters using the index of the count we
// reached before running out of removals.
const findLeastNumOfUniqueInts = (arr, k) => {
  // Pair each number with how frequently it occurs.
  const counts = new Map();
  for (const num of arr) {
    counts.set(num, counts.get(num) + 1 || 1);
  }

  // Sort the frequencies themselves. We don't need them tied to their number.
  const sortedCounts = [...counts.values()].toSorted((a, b) => a - b);

  for (let i = 0; i < sortedCounts.length; ++i) {
    // If we do not have enough removals for every occurrence of this number,
    // it and every number with >= frequency still remain.
    if (k < sortedCounts[i]) return sortedCounts.length - i;

    // Remove every instance of this number, decrementing k accordingly.
    k -= sortedCounts[i];
  }

  return 0;
};
```

// Approach: Greedily prioritize removing numbers with the lowest frequencies.
// We can track the number of unique characters with a single variable.
const findLeastNumOfUniqueInts = (arr, k) => {
  // Pair each number with how frequently it occurs
  const counts = new Map();
  for (const num of arr) {
    counts.set(num, counts.get(num) + 1 || 1);
  }

  // Sort the frequencies themselves. We don't need them tied to their number.
  const sortedCounts = [...counts.values()].toSorted((a, b) => a - b);

  // We start with all unique numbers found within the initial arr
  let uniqueNums = sortedCounts.length;

  // Iterate through each number (stopping early if k reaches <= 0)
  for (let i = 0; i < sortedCounts.length, k > 0; ++i) {
    // Subtract from k the frequency of this number
    k -= sortedCounts[i];

    // Can we remove all occurences of num? That removes 1 unique value from arr.
    if (k >= 0) --uniqueNums;
  }

  return uniqueNums;
};
```

// Approach: Greedily prioritize removing numbers with the lowest frequencies.
// We make use of a Set for easily tracking uniques.
const findLeastNumOfUniqueInts = (arr, k) => {
  const counts = new Map();
  for (const num of arr) {
    counts.set(num, counts.get(num) + 1 || 1);
  }

  const sortedCounts = [...counts.entries()].toSorted((a, b) => a[1] - b[1]);

  const set = new Set(arr);
  for (const [key, val] of sortedCounts) {
    if (val <= k) {
      k -= val;
      set.delete(key);
    } else break;
  }

  return set.size;
};
```
