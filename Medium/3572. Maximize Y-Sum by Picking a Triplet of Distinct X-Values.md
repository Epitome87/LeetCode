# [3572. Maximize Y-Sum by Picking a Triplet of Distinct X-Values](https://leetcode.com/problems/maximize-ysum-by-picking-a-triplet-of-distinct-xvalues)

## Description

You are given two integer arrays `x` and `y`, each of length `n`. You must choose three **distinct** indices `i`, `j`, and `k` such that:

- `x[i] != x[j]`
- `x[j] != x[k]`
- `x[k] != x[i]`

Your goal is to **maximize** the value of `y[i] + y[j] + y[k]` under these conditions. Return the **maximum** possible sum that can be obtained by choosing such a triplet of indices.

If no such triplet exists, return -1.

**Example 1:**

**Input:** x = \[1,2,1,3,2\], y = \[5,3,4,6,2\]  
**Output:** 14  
**Explanation:**

- Choose `i = 0` (`x[i] = 1`, `y[i] = 5`), `j = 1` (`x[j] = 2`, `y[j] = 3`), `k = 3` (`x[k] = 3`, `y[k] = 6`).
- All three values chosen from `x` are distinct. `5 + 3 + 6 = 14` is the maximum we can obtain. Hence, the output is 14.

**Example 2:**

**Input:** x = \[1,2,1,2\], y = \[4,5,6,7\]  
**Output:** \-1  
**Explanation:**

- There are only two distinct values in `x`. Hence, the output is -1.

**Constraints:**

- `n == x.length == y.length`
- `3 <= n <= 10^5`
- `1 <= x[i], y[i] <= 10^6`

## My Solution

**JavaScript**

```js
// Approach: Avoid sorting entire map of values, instead track the three largest values
const maxSumDistinctTriplet = (x, y) => {
  // Stores pairings between x[i] and y[i]
  // For repeated keys (x[i]), the maximum corresponding y[i] wins
  const map = new Map();
  for (let i = 0; i < x.length; ++i) {
    map.set(x[i], Math.max(map.get(x[i]) ?? 0, y[i]));
  }

  // 3 unique values are required for a solution
  if (map.size < 3) return -1;

  // Gather the three largest values
  let max1 = -Infinity;
  let max2 = -Infinity;
  let max3 = -Infinity;

  for (const val of map.values()) {
    if (val > max1) {
      max3 = max2;
      max2 = max1;
      max1 = val;
    } else if (val > max2) {
      max3 = max2;
      max2 = val;
    } else if (val > max3) {
      max3 = val;
    }
  }

  return max1 + max2 + max3;
};
```

| --------------- | ---------------- |
| O(N)            | O(N)             |

```js
const maxSumDistinctTriplet = (x, y) => {
  // Stores pairings between x[i] and y[i]
  // For repeated keys (x[i]), the maximum corresponding y[i] wins
  const map = new Map();
  for (let i = 0; i < x.length; ++i) {
    map.set(x[i], Math.max(map.get(x[i]) ?? 0, y[i]));
  }

  if (map.size < 3) return -1;

  // Gather and sort values found within the map, selecting the largest 3
  const values = Array.from(map.values());
  values.sort((a, b) => b - a);
  return values[0] + values[1] + values[2];
};
```

| O(N \* Log(N))  | O(N)             |
