# [3784. Minimum Deletion Cost to Make All Characters Equal](https://leetcode.com/problems/minimum-deletion-cost-to-make-all-characters-equal)

## Description

You are given a string `s` of length `n` and an integer array `cost` of the same length, where `cost[i]` is the cost to **delete** the `ith` character of `s`.

You may delete any number of characters from `s` (possibly none), such that the resulting string is **non-empty** and consists of **equal** characters.

Return an integer denoting the **minimum** total deletion cost required.

**Example 1:**

**Input:** s = "aabaac", cost = \[1,2,3,4,1,10\]  
**Output:** 11  
**Explanation:**  
Deleting the characters at indices 0, 1, 2, 3, 4 results in the string `"c"`, which consists of equal characters, and the total cost is `cost[0] + cost[1] + cost[2] + cost[3] + cost[4] = 1 + 2 + 3 + 4 + 1 = 11`.

**Example 2:**

**Input:** s = "abc", cost = \[10,5,8\]  
**Output:** 13  
**Explanation:**  
Deleting the characters at indices 1 and 2 results in the string `"a"`, which consists of equal characters, and the total cost is `cost[1] + cost[2] = 5 + 8 = 13`.

**Example 3:**

**Input:** s = "zzzzz", cost = \[67,67,67,67,67\]  
**Output:** 0  
**Explanation:**  
All characters in `s` are equal, so the deletion cost is 0.

**Constraints:**

- `n == s.length == cost.length`
- `1 <= n <= 10^5`
- `1 <= cost[i] <= 10^9`
- `s` consists of lowercase English letters.

## My Solution

**JavaScript**

```js
// Approach: One pass
const minCost = (s, cost) => {
  const charCost = Array(26).fill(0);
  let totalCost = 0;
  let maxCost = 0;

  for (let i = 0; i < cost.length; ++i) {
    totalCost += cost[i];
    const charIdx = s[i].charCodeAt() - 'a'.charCodeAt();
    charCost[charIdx] += cost[i];
    maxCost = Math.max(maxCost, charCost[charIdx]);
  }

  return totalCost - maxCost;
};
```

| --------------- | ---------------- |
| O(N)            | O(1)             |

```js
// Approach: Two pass
const minCost = (s, cost) => {
  // The cost of all occurrences of a character. charCost[0] is the cost of deleting all 'a',
  // charCost[25] of deleting all 'z', etc.
  const charCost = Array(26).fill(0);
  let totalCost = 0; // Total cost of every single character in s

  // One pass to calculate the total cost and the cost of each character.
  for (let i = 0; i < s.length; ++i) {
    const charIdx = s[i].charCodeAt() - 'a'.charCodeAt();
    totalCost += cost[i];
    charCost[charIdx] += cost[i];
  }

  // A second pass from a-z to find the best character to keep all occurrences of
  let result = Infinity;
  for (let i = 0; i < charCost.length; ++i) {
    // The cost to keep all occurrences of the current character (i.e delete all other characters)
    // is the total cost minus the cost of all occurrences of this current character.
    const costToKeep = totalCost - charCost[i];
    result = Math.min(result, costToKeep);
  }

  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N)            | O(1)             |
