# [462. Minimum Moves to Equal Array Elements II](https://leetcode.com/problems/minimum-moves-to-equal-array-elements-ii)

## Description

Given an integer array `nums` of size `n`, return _the minimum number of moves required to make all array elements equal_.

In one move, you can increment or decrement an element of the array by `1`.

Test cases are designed so that the answer will fit in a **32-bit** integer.

**Example 1:**

**Input:** nums = \[1,2,3\]  
**Output:** 2  
**Explanation:**  
Only two moves are needed (remember each move increments or decrements one element):
\[1,2,3\] => \[2,2,3\] => \[2,2,2\]

**Example 2:**

**Input:** nums = \[1,10,2,9\]  
**Output:** 16

**Constraints:**

- `n == nums.length`
- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

## My Solution

**JavaScript**

```js
// Approach: Sort the array.
// No need to calculate the median -- just take the difference between the element i from start with element i from end
const minMoves2 = (nums) => {
  let result = 0;
  const sorted = nums.toSorted((a, b) => a - b);

  for (let i = 0; i < sorted.length / 2; ++i) {
    result += sorted[sorted.length - i - 1] - sorted[i];
  }

  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N \* Log(N))  | O(Sorting)       |

```js
// Approach: Sort the array and calculate the median number. For each number in nums,
// determine the difference between it and the median.
const minMoves2 = (nums) => {
  let result = 0;
  const median = nums.toSorted((a, b) => a - b)[Math.floor(nums.length / 2)];

  for (const num of nums) {
    result += Math.abs(num - median);
  }

  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N \* Log(N))  | O(Sorting)       |
