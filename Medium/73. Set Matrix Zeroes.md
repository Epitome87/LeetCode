# [73. Set Matrix Zeroes](https://leetcode.com/problems/set-matrix-zeroes)

## Description

Given an `m x n` integer matrix `matrix`, if an element is `0`, set its entire row and column to `0`'s.

You must do it [in place](https://en.wikipedia.org/wiki/In-place_algorithm).

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/08/17/mat1.jpg)

**Input:** matrix = \[\[1,1,1\],\[1,0,1\],\[1,1,1\]\]  
**Output:** \[\[1,0,1\],\[0,0,0\],\[1,0,1\]\]

**Example 2:**

![](https://assets.leetcode.com/uploads/2020/08/17/mat2.jpg)

**Input:** matrix = \[\[0,1,2,0\],\[3,4,5,2\],\[1,3,1,5\]\]  
**Output:** \[\[0,0,0,0\],\[0,4,5,0\],\[0,3,1,0\]\]

**Constraints:**

- `m == matrix.length`
- `n == matrix[0].length`
- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

**Follow up:**

- A straightforward solution using `O(mn)` space is probably a bad idea.
- A simple improvement uses `O(m + n)` space, but still not the best solution.
- Could you devise a constant space solution?

## My Solution

**JavaScript**

```js
// Approach: Avoiding extra space by temporarily using the first row and first column to flag
// which rows / columns should be set to all 0's.
const setZeroes = (matrix) => {
  let modifyFirstRow = false;
  let modifyFirstCol = false;

  /* Step 1: Traverse the matrix. If we encounter any 0 in the first row or column,  we set
  the modifyFirstRow / modifyFirstCol to true, indicating that the 0th row / 0th col of the given
  matrix will become 0. If we encounter any 0, we will set the corresponding 0th row / 0th col values to 0, respectively. */
  for (let row = 0; row < matrix.length; row++) {
    for (let col = 0; col < matrix[0].length; col++) {
      if (matrix[row][col] === 0) {
        if (row === 0) modifyFirstRow = true;
        if (col === 0) modifyFirstCol = true;
        matrix[0][col] = 0;
        matrix[row][0] = 0;
      }
    }
  }

  // Step 2: Update the values of the matrix (skipping the 0th row and 0th col).
  // We update based on if the 0th row or 0th col of the current item is 0
  for (let row = 1; row < matrix.length; row++) {
    for (let col = 1; col < matrix[0].length; col++) {
      if (matrix[row][0] === 0 || matrix[0][col] === 0) {
        matrix[row][col] = 0;
      }
    }
  }

  // Step 3: At this point, all values in rows 1 and above, and cols 1 and above have their 0s set
  // But the 0th row and 0th col may need to be updated to 0 according to the other elements in their row / col.
  // We modify these cells last because they temporarily stored data that helped us fill out the rest of the matrix.
  if (modifyFirstRow) {
    for (let col = 0; col < matrix[0].length; col++) matrix[0][col] = 0;
  }
  if (modifyFirstCol) {
    for (let row = 0; row < matrix.length; row++) matrix[row][0] = 0;
  }
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(M \* N)       | O(1)             |

```js
// Approach: Using two Sets to track which rows & columns to modify
const setZeroes = (matrix) => {
  const modifiedRows = new Set();
  const modifiedCols = new Set();

  for (let r = 0; r < matrix.length; r++) {
    for (let c = 0; c < matrix[r].length; c++) {
      // If the cell is 0, flag this row and col for modification
      if (matrix[r][c] === 0) {
        modifiedRows.add(r);
        modifiedCols.add(c);
      }
    }
  }

  // Set all affected rows to 0
  for (let modifiedRow of modifiedRows) {
    for (let i = 0; i < matrix[modifiedRow].length; i++) {
      matrix[modifiedRow][i] = 0;
    }
  }

  // Set all affected columns to 0
  for (let row = 0; row < matrix.length; row++) {
    for (let modifiedCol of modifiedCols) {
      matrix[row][modifiedCol] = 0;
    }
  }
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(M \* N)       | O(M + N)         |
