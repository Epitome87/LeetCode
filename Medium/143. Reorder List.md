# [143. Reorder List](https://leetcode.com/problems/reorder-list)

## Description

You are given the head of a singly linked-list. The list can be represented as:

L0 → L1 → … → Ln - 1 → Ln

_Reorder the list to be on the following form:_

L0 → Ln → L1 → Ln - 1 → L2 → Ln - 2 → …

You may not modify the values in the list's nodes. Only nodes themselves may be changed.

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/03/04/reorder1linked-list.jpg)

**Input:** head = \[1,2,3,4\]  
**Output:** \[1,4,2,3\]

**Example 2:**

![](https://assets.leetcode.com/uploads/2021/03/09/reorder2-linked-list.jpg)

**Input:** head = \[1,2,3,4,5\]  
**Output:** \[1,5,2,4,3\]

**Constraints:**

- The number of nodes in the list is in the range `[1, 5 * 104]`.
- `1 <= Node.val <= 1000`

## My Solution

**JavaScript**

```js
const reorderList = (head) => {
  // Step 1: Find middle
  if (!head) return [];

  let slow = head;
  let fast = head;

  while (fast.next && fast.next.next) {
    slow = slow.next;
    fast = fast.next.next;
  }

  // Step 2: Reverse second half
  let prev = null;
  let cur = slow.next;

  while (cur) {
    let next = cur.next;
    cur.next = prev;
    prev = cur;
    cur = next;
  }

  slow.next = null;

  // Step 3: Merge lists alternately
  let head1 = head;
  let head2 = prev;

  while (head2) {
    let next = head1.next;
    head1.next = head2;
    head1 = head2;
    head2 = next;
  }
};
```

def reorderList(self, head: Optional[ListNode]) -> None:
    if not head or not head.next: return

    # Step 1: Split list into halves
    slow = head
    fast = head

    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next

    left_half = head
    right_half = slow.next

    current = right_half

        current = next

    head2 = right_half

        next1 = head1.next
        head1.next = head2

        head2 = next1

**Java**

```java
public void reorderList(ListNode head) {
    if (head == null || head.next == null) {
        return;
    }

    // Step 1: Divide list into two halves
    ListNode slow = head;
    ListNode fast = head;

    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }

    // Step 2: Reverse second half
    ListNode previous = null;
    ListNode current = slow.next;

    // Break connection between the two halves
    slow.next = null;

    while (current != null) {
        ListNode next = current.next;
        current.next = previous;
        previous = current;
        current = next;
    }

    // Step 3: Merge in zigzag pattern
    ListNode head1 = head;
    ListNode head2 = previous;

    while (head2 != null) {
        ListNode next1 = head1.next;
        head1.next = head2;
        head1 = head2;
        head2 = next1;
    }
}
```
