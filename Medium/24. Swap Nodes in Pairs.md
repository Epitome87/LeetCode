# [24. Swap Nodes in Pairs](https://leetcode.com/problems/swap-nodes-in-pairs)

## Description

Given a linked list, swap every two adjacent nodes and return its head. You must solve the problem without modifying the values in the list's nodes (i.e., only nodes themselves may be changed.)

**Example 1:**

**Input:** head = \[1,2,3,4\]  
**Output:** \[2,1,4,3\]

**Constraints:**

- The number of nodes in the list is in the range `[0, 100]`.
- `0 <= Node.val <= 100`

## My Solution

**JavaScript**

```js
const swapPairs = (head) => {
  // Base case: List is empty or only has one node? No swap possible!
  if (!head || !head.next) return head;

  // Create dummy node for easier return and edge cases
  const dummy = new ListNode();

  // Two pointers, one for tracking the node before the current pair,
  // the other tracking the first node of the current pair to swap
  let previous = dummy;
  let current = head;

  // Continue while we have at least two nodes left
  while (current && current.next) {
    // Connect previous node to the second node of the pair
    previous.next = current.next;

    // Save the node after the pair. Connect 1st node to this next node.
    current.next = previous.next.next;

    // Connect the second node to the first node
    previous.next.next = current;

    // Move the two pointers forward one step. Previous becomes the last node of the swapped pair.
    // Current moves to the start of the next pair.
    previous = current;
    current = current.next;
  }

  // Finally, return the new head!
  return dummy.next;
};
```

| --------------- | ---------------- |
| O(N)            | O(1)             |
