# [946. Validate Stack Sequences](https://leetcode.com/problems/validate-stack-sequences)

## Description

Given two integer arrays `pushed` and `popped` each with distinct values, return `true` _if this could have been the result of a sequence of push and pop operations on an initially empty stack, or_ `false` _otherwise._

**Example 1:**

**Input:** pushed = \[1,2,3,4,5\], popped = \[4,5,3,2,1\]  
**Output:** true  
**Explanation:** We might do the following sequence:  
push(1), push(2), push(3), push(4),  
pop() -> 4,  
push(5),  
pop() -> 5, pop() -> 3, pop() -> 2, pop() -> 1

**Example 2:**

**Input:** pushed = \[1,2,3,4,5\], popped = \[4,3,5,1,2\]  
**Output:** false  
**Explanation:** 1 cannot be popped before 2.

**Constraints:**

- `1 <= pushed.length <= 1000`
- `0 <= pushed[i] <= 1000`
- All the elements of `pushed` are **unique**.
- `popped.length == pushed.length`
- `popped` is a permutation of `pushed`.

## My Solution

**JavaScript**

```js
const validateStackSequences = (pushed, popped) => {
  const stack = [];
  let popIdx = 0;

  for (const num of pushed) {
    // Push each number in pushed into our simulation stack
    stack.push(num);

    // And while the last num added to the stack is the first in the popped array...
    while (stack.length && stack.at(-1) === popped[popIdx]) {
      // Pop the stack, and move on to the next element in popped array
      stack.pop();
      ++popIdx;
    }
  }

  // We have a valid pair of pushed / popped arrays if our simulated stack is empty
  return stack.length === 0;
};
```

| O(N)            | O(N)             |

**Python**

```python
def validateStackSequences(self, pushed: List[int], popped: List[int]) -> bool:
    stack = []
    popped_idx = 0

    for num in pushed:
        stack.append(num)

        while stack and stack[-1] == popped[popped_idx]:
            stack.pop()
            popped_idx += 1

    return len(stack) == 0
```

**Java**

```java
public boolean validateStackSequences(int[] pushed, int[] popped) {
    Deque<Integer> stack = new ArrayDeque();
    int popIdx = 0;

    for (int num : pushed) {
        stack.push(num);

        while (!stack.isEmpty() && popped[popIdx] == stack.peek()) {
            stack.pop();
            popIdx++;
        }
    }

    return stack.isEmpty();
}
```
