# [17. Letter Combinations of a Phone Number](https://leetcode.com/problems/letter-combinations-of-a-phone-number)

## Description

Given a string containing digits from `2-9` inclusive, return all possible letter combinations that the number could represent. Return the answer in **any order**.

A mapping of digits to letters (just like on the telephone buttons) is given below. Note that 1 does not map to any letters.

![](https://assets.leetcode.com/uploads/2022/03/15/1200px-telephone-keypad2svg.png)

**Example 1:**

**Input:** digits = "23"  
**Output:** \["ad","ae","af","bd","be","bf","cd","ce","cf"\]

**Example 2:**

**Input:** digits = "2"  
**Output:** \["a","b","c"\]

**Constraints:**

- `1 <= digits.length <= 4`
- `digits[i]` is a digit in the range `['2', '9']`.

## My Solution

**JavaScript**

```js
const letterCombinations = (digits) => {
  const digitToChars = { 2: 'abc', 3: 'def', 4: 'ghi', 5: 'jkl', 6: 'mno', 7: 'pqrs', 8: 'tuv', 9: 'wxyz' };
  const result = [];

  const backtrack = (idx, prefix) => {
    if (idx === digits.length) {
      result.push(prefix);
      return;
    }

    // For this digit at idx, explore each character that is mapped to it
    for (const char of digitToChars[digits[idx]]) {
      // Choose this current character
      backtrack(idx + 1, prefix + char);
    }
  };

  backtrack(0, '');
  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(4^N)          | O(N)             |

def letterCombinations(self, digits: str) -> List[str]:
    digit_to_chars = { '2': 'abc', '3': 'def', '4': 'ghi', '5': 'jkl', '6': 'mno', '7': 'pqrs', '8': 'tuv', '9': 'wxyz' }
    result = []

    def backtrack(idx, prefix):
        if idx == len(digits):
            result.append(prefix)
            return

        for char in digit_to_chars[digits[idx]]:
            backtrack(idx + 1, prefix + char)

    backtrack(0, '')
    return result

**Java**

```java
class Solution {
    public List<String> letterCombinations(String digits) {
        List<String> result = new ArrayList<>();
        Map<Character, String> digitToChars = Map.of('2', "abc", '3', "def", '4', "ghi", '5', "jkl",
            '6', "mno", '7', "pqrs", '8', "tuv", '9', "wxyz");

        backtrack(digits, 0, new StringBuilder(), digitToChars, result);
        return result;
    }

    private void backtrack(String digits, int idx, StringBuilder prefix, Map<Character, String> digitToChars, List<String> result) {
        if (idx == digits.length()) {
            result.add(prefix.toString());
            return;
        }

        char digit = digits.charAt(idx);
        String letters = digitToChars.get(digit);

        for (char ch : letters.toCharArray()) {
            prefix.append(ch);
            backtrack(digits, idx + 1, prefix, digitToChars, result);
            prefix.deleteCharAt(prefix.length() - 1);
        }
    }
}
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(4^N)          | O(N)             |
