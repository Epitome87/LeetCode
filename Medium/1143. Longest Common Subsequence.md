# [1143. Longest Common Subsequence](https://leetcode.com/problems/longest-common-subsequence)

## Description

Given two strings `text1` and `text2`, return _the length of their longest **common subsequence**._ If there is no **common subsequence**, return `0`.

A **subsequence** of a string is a new string generated from the original string with some characters (can be none) deleted without changing the relative order of the remaining characters.

- For example, `"ace"` is a subsequence of `"abcde"`.

A **common subsequence** of two strings is a subsequence that is common to both strings.

**Example 1:**

**Input:** text1 = "abcde", text2 = "ace"  
**Output:** 3  
**Explanation:** The longest common subsequence is "ace" and its length is 3.

**Example 2:**

**Input:** text1 = "abc", text2 = "abc"  
**Output:** 3  
**Explanation:** The longest common subsequence is "abc" and its length is 3.

**Example 3:**

**Input:** text1 = "abc", text2 = "def"  
**Output:** 0  
**Explanation:** There is no such common subsequence, so the result is 0.

**Constraints:**

- `1 <= text1.length, text2.length <= 1000`
- `text1` and `text2` consist of only lowercase English characters.

## My Solution

**JavaScript**

```js
// Approach: Bottom-up Dynamic Programming (Tabulation), space optimized!
const longestCommonSubsequence = (text1, text2) => {
  if (text1.length < text2.length) {
    [text1, text2] = [text2, text1];
  }

  const m = text1.length; // The longer (or equal length) string
  const n = text2.length; // The shorter (or equal length) string

  // We only need a 1D array, opting for columns for the shorter string
  let prev = new Array(n + 1).fill(0);

  for (let i = 1; i <= m; ++i) {
    // Initialize a new row
    const cur = new Array(n + 1).fill(0);

    for (let j = 1; j <= n; ++j) {
      // Scenario 1: Characters are equal
      if (text1[i - 1] === text2[j - 1]) {
        // The previous best is when BOTH strings had one less char
        // I.e diagonal from previous row. Here prev is the previous (and only) row.
        cur[j] = 1 + prev[j - 1];
      } else {
        // Scenario 2: Characters are not equal
        // The result is either i - 1 or j - 1
        cur[j] = Math.max(prev[j], cur[j - 1]);
      }
    }

    prev = cur;
  }

  return prev[n];
};
```

// Approach: Bottom-up Dynamic Programming (Tabulation)
const longestCommonSubsequence = (text1, text2) => {
  const m = text1.length;
  const n = text2.length;
  const dp = Array.from({ length: m + 1 }, () => new Array(n + 1).fill(0));

  // These are already 0 because of fill(0), but being explicit for clarity
  for (let i = 0; i <= m; i++) dp[i][0] = 0;
  for (let j = 0; j <= n; j++) dp[0][j] = 0;

  for (let i = 1; i <= m; ++i) {
    for (let j = 1; j <= n; ++j) {
      // Scenario 1: Characters are equal
      if (text1[i - 1] === text2[j - 1]) {
        // The previous best is when BOTH strings had one less char
        dp[i][j] = 1 + dp[i - 1][j - 1];
      } else {
        // Scenario 2: Characters are not equal
        // The result is either i - 1 or j - 1
        dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
      }
    }
  }

  return dp[m][n];
};

```js
// Approach: Dynamic Programming (via recursion) & memoization
// Note: Without memoization, time complexity is O(2 ^ (M + N))
const longestCommonSubsequence = (text1, text2) => {
  const memo = {};

  const dp = (i, j) => {
    // If either of the strings are empty, we've reached a base case
    if (i === 0 || j === 0) return 0;

    const key = i + ',' + j;
    if (key in memo) return memo[key];

    // If characters are equal, move onto the next char of both strings
    if (text1[i - 1] === text2[j - 1]) {
      memo[key] = 1 + dp(i - 1, j - 1);
      return memo[key];
    }

    // Otherwise, find whether skipping the char in text1 or skipping the char in text2
    // leads to a greater subsequence.
    const skip1 = dp(i - 1, j);
    const skip2 = dp(i, j - 1);

    memo[key] = Math.max(skip1, skip2);
    return memo[key];
  };

  return dp(text1.length, text2.length);
};
```

| O(M \* N)       | O(M \* N)        |
