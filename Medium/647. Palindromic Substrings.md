# [647. Palindromic Substrings](https://leetcode.com/problems/palindromic-substrings)

## Description

Given a string `s`, return _the number of **palindromic substrings** in it_.

A string is a **palindrome** when it reads the same backward as forward.

A **substring** is a contiguous sequence of characters within the string.

**Example 1:**

**Input:** s = "abc"  
**Output:** 3  
**Explanation:** Three palindromic strings: "a", "b", "c".

**Example 2:**

**Input:** s = "aaa"  
**Output:** 6  
**Explanation:** Six palindromic strings: "a", "a", "a", "aa", "aa", "aaa".

**Constraints:**

- `1 <= s.length <= 1000`
- `s` consists of lowercase English letters.

## My Solution

**JavaScript**

```js
// Approach: Treat each character as the center of a palindrome and check.
// O(N^2) time and O(1) space
const countSubstrings = (s) => {
  let result = 0;

  for (let i = 0; i < s.length; ++i) {
    // Center at i for odd-length palindromes
    result += expandAroundCenter(s, i, i);

    // Center between i and i + 1 for even-length palindromes
    result += expandAroundCenter(s, i, i + 1);
  }

  return result;
};

// Check how many substrings of s are palindromic
const expandAroundCenter = (s, left, right) => {
  let count = 0;

  while (left >= 0 && right < s.length && s[left] === s[right]) {
    ++count;
    --left;
    ++right;
  }

  return count;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N^2)          | O(1)             |
