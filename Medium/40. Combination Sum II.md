# [40. Combination Sum II](https://leetcode.com/problems/combination-sum-ii)

## Description

Given a collection of candidate numbers (`candidates`) and a target number (`target`), find all unique combinations in `candidates` where the candidate numbers sum to `target`.

Each number in `candidates` may only be used **once** in the combination.

**Note:** The solution set must not contain duplicate combinations.

**Example 1:**

**Input:** candidates = \[10,1,2,7,6,1,5\], target = 8  
**Output:**  
\[  
\[1,1,6\],  
\[1,2,5\],  
\[1,7\],  
\[2,6\]  
\]

**Example 2:**

**Input:** candidates = \[2,5,2,1,2\], target = 5  
**Output:**  
\[  
\[1,2,2\],  
\[5\]  
\]

**Constraints:**

- `1 <= candidates.length <= 100`
- `1 <= candidates[i] <= 50`
- `1 <= target <= 30`

## My Solution

**JavaScript**

```js
// Approach: Backtracking
const combinationSum2 = (candidates, target) => {
  const result = [];
  const curCombination = [];

  // Sort the array to help with early returning and detecting duplicate combinations
  candidates.sort((a, b) => a - b);

  const backtrack = (sum, startIdx) => {
    if (sum === target) {
      // Combination sum was found: add it as a result and stop recursing
      result.push([...curCombination]);
      return;
    }

    for (let i = startIdx; i < candidates.length; ++i) {
      // Pruning 1: Skip duplicates at the same tree level. This prevents
      // generating identical combinations, i.e [1, 1, 2] and [1, 1, 2]
      if (i > startIdx && candidates[i] === candidates[i - 1]) continue;

      // Pruning 2: Stop exploring once candidates sum past the target
      if (sum + candidates[i] > target) break;

      // Choose the current number
      curCombination.push(candidates[i]);

      // Explore (And move to next idx, since we can't re-use same element)
      backtrack(sum + candidates[i], i + 1);

      // Unchoose (backtrack)
      curCombination.pop();
    }
  };

  backtrack(0, 0);
  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(2^N)          | O(N)             |
