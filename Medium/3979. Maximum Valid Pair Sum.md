# [3979. Maximum Valid Pair Sum](https://leetcode.com/problems/maximum-valid-pair-sum)

## Description

You are given an integer array `nums` of length `n` and an integer `k`.

A pair of indices `(i, j)` is called **valid** if:

- `0 <= i < j < n`
- `j - i >= k`

Return the **maximum** value of `nums[i] + nums[j]` among all valid pairs.

**Example 1:**

**Input:** nums = \[1,3,5,2,8\], k = 2  
**Output:** 13  
**Explanation:**

The valid pairs are:

- `(0, 2)`: `nums[0] + nums[2] = 6`
- `(0, 3)`: `nums[0] + nums[3] = 3`
- `(0, 4)`: `nums[0] + nums[4] = 9`
- `(1, 3)`: `nums[1] + nums[3] = 5`
- `(1, 4)`: `nums[1] + nums[4] = 11`
- `(2, 4)`: `nums[2] + nums[4] = 13`

Thus, the answer is 13.​​​​​​​

**Example 2:**

**Input:** nums = \[5,1,9\], k = 1  
**Output:** 14  
**Explanation:**

- Since `k = 1`, every pair is valid.
- The maximum value is obtained from a pair `(0, 2)`​​​​​​​, which is `nums[0] + nums[2] = 5 + 9 = 14`.
- Thus, the answer is 14.

**Constraints:**

- `2 <= n == nums.length <= 10^5`
- `1 <= nums[i] <= 10^9`
- `1 <= k <= n - 1`

## My Solution

**JavaScript**

```js
// Approach: Start iterating from i = k. At every iteration, we see what the maximum of the "tail" is.
// That is, the maximum out of all the elements that are >= k elements away from the current element.
// Given that tail maximum, we update our result to check if the tail max + the current element produces
// a new global maximum.
const maxValidPairSum = (nums, k) => {
  let result = 0;
  let tailMax = 0;

  for (let i = k; i < nums.length; ++i) {
    tailMax = Math.max(tailMax, nums[i - k]);
    result = Math.max(result, nums[i] + tailMax);
  }

  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N)            | O(1)             |
