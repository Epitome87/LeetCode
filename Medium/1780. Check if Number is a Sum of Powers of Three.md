# [1780. Check if Number is a Sum of Powers of Three](https://leetcode.com/problems/check-if-number-is-a-sum-of-powers-of-three)

## Description

Given an integer `n`, return `true` _if it is possible to represent_ `n` _as the sum of distinct powers of three._ Otherwise, return `false`.

An integer `y` is a power of three if there exists an integer `x` such that `y == 3x`.

**Example 1:**

**Input:** n = 12  
**Output:** true  
**Explanation:** 12 = 31 + 32

**Example 2:**

**Input:** n = 91  
**Output:** true  
**Explanation:** 91 = 30 + 32 + 34

**Example 3:**

**Input:** n = 21  
**Output:** false

**Constraints:**

- `1 <= n <= 10^7`

## My Solution

**JavaScript**

```js
// Approach: Convert the number to base 3. A valid solution is one where
// each possible digit in a base 3 system (0, 1, or 2) is either 0 or 1,
// representing no power of 3 for that exponent or a single power of 3.
const checkPowersOfThree = (n) => {
  while (n) {
    if (n % 3 === 2) return false;
    n = Math.floor(n / 3);
  }

  return true;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(Log(N))       | O(1)             |

**Python**

```python
def checkPowersOfThree(self, n: int) -> bool:
    while n:
        if n % 3 == 2: return False
        n //= 3
    return True
```

| --------------- | ---------------- |
| O(Log(N))       | O(1)             |
