# [3211. Generate Binary Strings Without Adjacent Zeros](https://leetcode.com/problems/generate-binary-strings-without-adjacent-zeros)

## Description

You are given a positive integer `n`.

A binary string `x` is **valid** if all substrings of `x` of length 2 contain **at least** one `"1"`.

Return all **valid** strings with length `n`**,** in _any_ order.

**Example 1:**

**Input:** n = 3  
**Output:** \["010","011","101","110","111"\]  
**Explanation:**  
The valid strings of length 3 are: `"010"`, `"011"`, `"101"`, `"110"`, and `"111"`.

**Example 2:**

**Input:** n = 1  
**Output:** \["0","1"\]  
**Explanation:**  
The valid strings of length 1 are: `"0"` and `"1"`.

**Constraints:**

- `1 <= n <= 18`

## My Solution

**JavaScript**

```js
const validStrings = (n) => {
  let result = [];

  const backtrack = (numDigits, prefix) => {
    // We've generated a new valid binary string of length n
    if (numDigits >= n) {
      result.push(prefix);
      return;
    }

    // Regardless of the last digit, 1 can always be an option
    backtrack(numDigits + 1, prefix + '1');

    // 0 can only be an option if last digit is not a 0
    if (prefix.at(-1) !== '0') {
      backtrack(numDigits + 1, prefix + '0');
    }
  };

  backtrack(0, '');
  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(2^N)          | O(N)             |
