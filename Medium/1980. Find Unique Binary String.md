# [1980. Find Unique Binary String](https://leetcode.com/problems/find-unique-binary-string)

## Description

Given an array of strings `nums` containing `n` **unique** binary strings each of length `n`, return _a binary string of length_ `n` _that **does not appear** in_ `nums`_. If there are multiple answers, you may return **any** of them_.

**Example 1:**

**Input:** nums = ["01","10"]  
**Output:** "11"  
**Explanation:** "11" does not appear in nums. "00" would also be correct.

**Example 2:**

**Input:** nums = ["00","01"]  
**Output:** "11"  
**Explanation:** "11" does not appear in nums. "10" would also be correct.

**Example 3:**

**Input:** nums = ["111","011","001"]  
**Output:** "101"  
**Explanation:** "101" does not appear in nums. "000", "010", "100", and "110" would also be correct.

**Constraints:**

- `n == nums.length`
- `1 <= n <= 16`
- `nums[i].length == n`
- `nums[i]` is either `'0'` or `'1'`.
- All the strings of `nums` are **unique**.

## My Solution

**JavaScript**

```js
// Approach: Using the concept of "Cantor's Diagonal".
// Essentially, if we line up each number in a series of rows and columns,
// we can produce an entirely unique number if we change each digit diagonally.
// This works due to the constraint that nums includes n strings of length n.
const findDifferentBinaryString = (nums) => {
  let result = [];

  for (let i = 0; i < nums.length; ++i) {
    result.push(nums[i][i] === '0' ? '1' : '0');
  }

  return result.join('');
};
```

| --------------- | ---------------- |
| O(N)            | O(1)             |

```js
// Approach: Using Backtracking to generate possible binary strings and
// return one not present in the nums array (turned into a set for O(1) look-up).
const findDifferentBinaryString = (nums) => {
  const set = new Set(nums);
  let result = '';

  const solve = (binary) => {
    if (binary.length === nums.length) {
      if (!set.has(binary)) {
        result = binary;
        return;
      }
      return '';
    }

    // Choose to add '0' to string
    solve(binary + '0');

    // Choose to add '1' to string
    solve(binary + '1');
  };

  solve('');
  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(2^N)          | O(N)             |
