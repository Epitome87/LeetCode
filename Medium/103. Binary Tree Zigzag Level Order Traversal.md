# [103. Binary Tree Zigzag Level Order Traversal](https://leetcode.com/problems/binary-tree-zigzag-level-order-traversal)

## Description

Given the `root` of a binary tree, return _the zigzag level order traversal of its nodes' values_. (i.e., from left to right, then right to left for the next level and alternate between).

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/02/19/tree1.jpg)

**Input:** root = \[3,9,20,null,null,15,7\]  
**Output:** \[\[3\],\[20,9\],\[15,7\]\]

**Example 2:**

**Input:** root = \[1\]  
**Output:** \[\[1\]\]

**Example 3:**

**Input:** root = \[\]  
**Output:** \[\]

**Constraints:**

- The number of nodes in the tree is in the range `[0, 2000]`.
- `-100 <= Node.val <= 100`

## My Solution

**JavaScript**

```js
// Approach: Rather than reversing every other level, create a fixed-size array for
// each level and use the complement of index i when going from right-to-left.
const zigzagLevelOrder = (root) => {
  if (!root) return [];

  const result = [];
  let isLeftward = false;
  const queue = [root];

  while (queue.length) {
    const levelSize = queue.length;
    const levelNodes = Array(levelSize);

    for (let i = 0; i < levelSize; ++i) {
      const node = queue.shift();

      const idx = isLeftward ? levelSize - i - 1 : i;
      levelNodes[idx] = node.val;

      if (node.left) queue.push(node.left);
      if (node.right) queue.push(node.right);
    }

    result.push(levelNodes);
    isLeftward = !isLeftward;
  }

  return result;
};
```

```js
// Approach: BFS, for no real reason!
const zigzagLevelOrder = (root) => {
  if (!root) return [];

  const result = [];

  const dfs = (node, level) => {
    if (!node) return;

    if (level >= result.length) result.push([]);

    // We ziggin', or we zaggin'?
    if (level % 2) result[level].unshift(node.val);
    else result[level].push(node.val);

    dfs(node.left, level + 1);
    dfs(node.right, level + 1);
  };

  dfs(root, 0);
  return result;
};
```

**Python**

```python
def zigzagLevelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
    if not root: return []

    result = []
    queue = deque([root])
    level = 0

    while queue:
        level_size = len(queue)
        level_nodes = [None] * level_size

        for i in range(level_size):
            node = queue.popleft()
            idx = level_size - i - 1 if level % 2 else i
            level_nodes[idx] = node.val
            if node.left: queue.append(node.left)
            if node.right: queue.append(node.right)

        result.append(level_nodes)
        level += 1

    return result
```
