# [82. Remove Duplicates from Sorted List II](https://leetcode.com/problems/remove-duplicates-from-sorted-list-ii)

## Description

Given the `head` of a sorted linked list, _delete all nodes that have duplicate numbers, leaving only distinct numbers from the original list_. Return _the linked list **sorted** as well_.

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/01/04/linkedlist1.jpg)

**Input:** head = \[1,2,3,3,4,4,5\]  
**Output:** \[1,2,5\]

**Example 2:**

![](https://assets.leetcode.com/uploads/2021/01/04/linkedlist2.jpg)

**Input:** head = \[1,1,1,2,3\]  
**Output:** \[2,3\]

**Constraints:**

- The number of nodes in the list is in the range `[0, 300]`.
- `-100 <= Node.val <= 100`
- The list is guaranteed to be **sorted** in ascending order.

## My Solution

**JavaScript**

```js
const deleteDuplicates = (head) => {
  // Dummy node for easier return and edge cases
  let dummy = new ListNode(0, head);
  let lastNode = dummy; // Points to the last known unique node

  while (head && head.next) {
    if (head.val === head.next.val) {
      // Skip nodes with the same value as current
      while (head && head.next && head.val === head.next.val) {
        head = head.next;
      }
      // By this point, head points to the last duplicate value in the sequence,
      // so we increment head to point to the first non-duplicate
      head = head.next;

      // Connect the last unique node to the new unique node
      lastNode.next = head;
    } else {
      // Update our last Node to point to its next reference
      lastNode = lastNode.next;

      // Move onto the next Node
      head = head.next;
    }
  }

  return dummy.next;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N)            | O(1)             |
