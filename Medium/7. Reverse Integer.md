# [7. Reverse Integer](https://leetcode.com/problems/reverse-integer)

## Description

Given a signed 32-bit integer `x`, return `x` _with its digits reversed_. If reversing `x` causes the value to go outside the signed 32-bit integer range `[-2^31, 2^31 - 1]`, then return `0`.

**Assume the environment does not allow you to store 64-bit integers (signed or unsigned).**

**Example 1:**

**Input:** x = 123  
**Output:** 321

**Example 2:**

**Input:** x = -123  
**Output:** -321

**Example 3:**

**Input:** x = 120  
**Output:** 21

**Constraints:**

- `-2^31 <= x <= 2^31 - 1`

## My Solution

**JavaScript**

```js
const reverse = (x) => {
  let result = 0;
  const INT_MAX = 2147483647;
  const INT_MIN = -2147483648;

  while (x !== 0) {
    const digit = x % 10;
    const nextResult = result * 10 + digit;

    // This works in JS because numbers are over 32 bits
    if (nextResult > INT_MAX || nextResult < INT_MIN) {
      return 0;
    }

    result = nextResult;
    x = Math.trunc(x / 10); // Key logic! Floor will not work properly for negatives
  }

  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(Log(X))       | O(1)             |
