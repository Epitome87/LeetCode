# [413. Arithmetic Slices](https://leetcode.com/problems/arithmetic-slices)

## Description

An integer array is called arithmetic if it consists of **at least three elements** and if the difference between any two consecutive elements is the same.

- For example, `[1,3,5,7,9]`, `[7,7,7,7]`, and `[3,-1,-5,-9]` are arithmetic sequences.

Given an integer array `nums`, return _the number of arithmetic **subarrays** of_ `nums`.

A **subarray** is a contiguous subsequence of the array.

**Example 1:**

**Input:** nums = \[1,2,3,4\]  
**Output:** 3  
**Explanation:** We have 3 arithmetic slices in nums: \[1, 2, 3\], \[2, 3, 4\] and \[1,2,3,4\] itself.

**Example 2:**

**Input:** nums = \[1\]  
**Output:** 0

**Constraints:**

- `1 <= nums.length <= 5000`
- `-1000 <= nums[i] <= 1000`

## My Solution

**JavaScript**

```js
// Approach: Keep track of current streak rather than the start index of each streak.
const numberOfArithmeticSlices = (nums) => {
  const n = nums.length;
  if (n < 3) return 0;

  let result = 0;
  let currentStreak = 0;

  for (let i = 2; i < n; ++i) {
    if (nums[i] - nums[i - 1] === nums[i - 1] - nums[i - 2]) {
      ++currentStreak;
      result += currentStreak;
    } else currentStreak = 0;
  }

  return result;
};
```

```js
const numberOfArithmeticSlices = (nums) => {
  const n = nums.length;
  if (n < 3) return 0;

  let count = 0;
  let startIdx = 0;
  let targetDiff = nums[0] - nums[1];

  for (let i = 1; i < nums.length; ++i) {
    const diff = nums[i - 1] - nums[i];

    // If current subarray has at least 3 elements and the difference
    // between this value and the previous is the same as our target difference...
    if (diff === targetDiff && i - startIdx >= 2) {
      // The length of the current run is L = i - startIdx + 1
      // The number of NEW slices ending at i is L - 2
      // The first two elements cannot be part of the run, as length must be >= 3
      count += i - startIdx - 1;
    }

    if (diff !== targetDiff) {
      // Mark nums[i - 1] and nums[i] as the potential start of a new arithmetic subarray
      startIdx = i - 1;
      // And their difference becomes the new target difference
      targetDiff = diff;
    }
  }

  return count;
};
```

| --------------- | ---------------- |
| O(N)            | O(1)             |

```python
def numberOfArithmeticSlices(self, nums: List[int]) -> int:
    n = len(nums)
    if n < 3: return 0

    result = 0
    current_streak = 0

    for i in range(2, n):
        if nums[i] - nums[i - 1] == nums[i - 1] - nums[i - 2]:
            current_streak += 1
            result += current_streak
        else:
            current_streak = 0

    return result
```
