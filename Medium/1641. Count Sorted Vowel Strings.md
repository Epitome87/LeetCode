# [1641. Count Sorted Vowel Strings](https://leetcode.com/problems/count-sorted-vowel-strings)

## Description

Given an integer `n`, return _the number of strings of length_ `n` _that consist only of vowels (_`a`_,_ `e`_,_ `i`_,_ `o`_,_ `u`_) and are **lexicographically sorted**._

A string `s` is **lexicographically sorted** if for all valid `i`, `s[i]` is the same as or comes before `s[i+1]` in the alphabet.

**Example 1:**

**Input:** n = 1  
**Output:** 5  
**Explanation:** The 5 sorted strings that consist of vowels only are `["a","e","i","o","u"].`

**Example 2:**

**Input:** n = 2  
**Output:** 15  
**Explanation:** The 15 sorted strings that consist of vowels only are
\["aa","ae","ai","ao","au","ee","ei","eo","eu","ii","io","iu","oo","ou","uu"\].  
Note that "ea" is not a valid string since 'e' comes after 'a' in the alphabet.

**Example 3:**

**Input:** n = 33  
**Output:** 66045

**Constraints:**

- `1 <= n <= 50`

## My Solution

**JavaScript**

```js
// Approach: Math! Using "Bars and Stars" method of counting.
// We have n stars (string length) and 4 bars (number of vowels minus 1).
// This is the same as (n + 4) choose (4). The formula for m choose k is:
// m! / (k! * (m - k)!). In our case this simplifies to (n + 4)! / (4!n!)
// To drop the factorial, we can simplify to:
// (n + 4)(n + 3)(n + 2)(n + 1)n!  /  24n!
// The n! terms cancel out, so (n + 4)(n + 3)(n + 2)(n + 1) / 24
const countVowelStrings = (n) => ((n + 4) * (n + 3) * (n + 2) * (n + 1)) / 24;
```

```js
// Approach: Dynamic Programming. Observe that the number of valid strings of
// length 2 ending in 'e' is equal to (number of length 1 strings ending in 'a') +
// (number of length 1 strings ending in 'e').
const countVowelStrings = (n) => {
  // Represents the counts of strings ending in [a, e, i, o, u]
  // For n = 1, there is one count for each vowel
  const counts = [1, 1, 1, 1, 1];

  // Already pre-calculated n = 1, so iterate n - 1 times
  for (let i = 1; i < n; ++i) {
    // Note no counts[0] modification, b/c 'a' can only follow 'a'
    // Thus counts[0] stays the same

    // 'e' can follow a or e
    counts[1] += counts[0];

    // 'i' can follow a, e, or i
    counts[2] += counts[1];

    // 'o' can follow a, e, i, or o
    counts[3] += counts[2];

    // 'u' can come after any vowel
    counts[4] += counts[3];
  }

  return counts.reduce((acc, cur) => acc + cur, 0);
};
```

```js
// Approach: Straightforward backtracking
const countVowelStrings = (n) => {
  let result = 0;
  const VOWEL_COUNT = 5;

  const backtrack = (n, prevVowelIdx) => {
    // Once we've constructed a string of length n, a result is formed!
    if (n === 0) {
      ++result;
      return;
    }

    // We can select the previous vowel again, or one of the ones after it
    for (let i = prevVowelIdx; i < VOWEL_COUNT; ++i) {
      backtrack(n - 1, i);
    }
  };

  backtrack(n, 0);
  return result;
};
```

| Time Complexity | Space Complexity |
| --------------- | ---------------- |
| O(N^4)          | O(N)             |
